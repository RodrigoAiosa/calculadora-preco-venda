# Calculadora de Preço de Venda

Aplicação desenvolvida em Python com Streamlit para cálculo de preço ideal de venda considerando:

- Custo do produto
- Custos adicionais
- Margem de lucro
- Impostos
- Comissões
- Taxas de pagamento

Fórmula utilizada:

Preço de Venda = Custo Total / (1 - Percentual Total)

Para rodar localmente:

pip install -r requirements.txt
streamlit run main.py
